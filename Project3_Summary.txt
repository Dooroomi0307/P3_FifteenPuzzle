Our project is an interactive puzzle game using javascript and CSS.
Implementing a proper fifteen puzzle algorithm with JavaScript to provide responsive and interactive game interface with extra features that enhance the user experience. 
Key Components/Challenges:
- Dividing a single image into 16 separated images
- Shuffling algorithm
- Tracking empty space
- Detecting an empty space around clicked tile
